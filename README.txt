# VisionTechApp

## Descripción
Este es un sistema de reconocimiento facial desarrollado para VisionTech Solutions. Utiliza Python y varias librerías para proporcionar funcionalidades de autenticación mediante reconocimiento facial.

## Requisitos
1. **Python 3.8 o superior**.
2. **Librerías**: Necesitas instalar las siguientes dependencias.

## Instalación del Entorno Virtual
